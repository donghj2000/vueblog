# vueblog
基于Vue3.2.x, element-plus 2.x的博客前端，包括注册/登录，个人信息修改，文章列表，评论，全文搜索，文章后台管理界面等，支持markdown编辑器。
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
